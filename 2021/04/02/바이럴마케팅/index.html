2. 통합 바이럴마케팅

  • 디지털 소비자의 성격 : 구매 정보 탐색에 적극적 (일상적 추적 과정에서 무의식적으로 브랜드 호감도와 구매결정이 되어있기 때문에 인식을 널리 하는 것이 중요)

  • ZMOT : 과거에는 first moment가 매장에서 처음 제품을 만났으나 이제는 그 전에 제품을 만나는 순간들이 있다. 이를 Zero Moment of Truth 라고 한다.

    • couch search : 마치 소파에 앉아 스포츠를 보는 것 처럼 열정적으로 서칭한다는 의미

  • SEO (Search Engine Optimization) : 검색 엔진 최적화로 소비자들의 구매행동을 유도하는 작업

    키워드 전략 및 고퀄리티 컨텐츠를 제작하여 소비자 궁금증을 해소할 수 있는 정보들 시딩

    • SNS / SEO / PR / Influencers / Alliance

  • 캠페인 액션 플랜 (순서 없이 마구잡이로 하면X 각 매체별 액션 플랜을 참고해서 진행)

    • 진행매체 구성 ( 이전에 전체적인 그림과 키워드를 미리 생각해두는 것이 좋음)

      1. 멀티 채널 구성

      2. 채널 별 리딩 전략

    • 공략키워드

      1. Core + Sub Keywords

      2. 컨텐츠 생성 수 / 검색략

    • 컨텐츠 컨셉

      1. 카테고리별로, 타겟별로 설정

      2. 맞춤화 / 개인화

    • 스케쥴링 (스케쥴에 따라 전체적은 운영 계획이 달라지기 때문에 기획 필요)

      1. 캠페인 관점의 스케쥴

      2. 컨텐츠 운영의 스케쥴

  • 공략키워드

    • 카테고리 키워드

      Need Recognition : 아 이거 찾아봐야겠다~ 이런게 있네?

      Pre-Search : 오 이거 괜찮네!

    • 브랜드/프로덕트 키워드

      Evalutaion : 평가하다 컨텐츠 불충분하네? 다시 Pre-Search로 돌아가자

      Search : 흥미가 생기네 더 찾아보자

  • 인플루언서 마케팅

    • 밀레니얼 세대들은 디지털 기기를 활용하여 정보 얻은 후 서비스 소비
    • 소셜 인플루 엇너의 커지는 영향력 (연예인에게 느끼는 친밀감보다 약 7배 높다)
    • 섭외 > 제작 > 노출
