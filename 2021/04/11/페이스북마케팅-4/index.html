1. 페이스북 광고 집행하기

2. 크리에이트 요소별 제작 노하우

  1. 텍스트
  • 간결하게 쓰기
    • 노출 위치별 미리보기 확인
    • 데스크톱/모바일 버전 따로 운영
    • 문구 80자 이하 인게이지먼트 27% 향상
  • 후킹하게 쓰기
    • Pain point - Solution
    • 제품 혜택, 장점
    • 해시태그 활용(최대30개)
    • 인게이지먼트 높은 단어
  • 수치활용하기
    • 특정 기간에 팔린 제품수량
    • 고객 리뷰수
    • 할인율 1+1 00% 할인
    • 말보다 수치
  • CTA 포함
    • 동사형 CTA
    • Facebook에서 제공하는 CTA버튼 활용(CTR최대 2.85배 상승)
    • 패션/뷰티 : SHOP NOW

  1. 영상
  • 권장길이 최대 15초, 최대 47%가 첫 3초 내에 전달, 최대 74%가 10초 이내
  • 인스타그램 스토리 비 9:16 형식 지원(몰입도 증가)
  • 컨셉과 스토리텔링의 방향성 정할 것
  • 소리 없이 이해할 수 있는 자막, 썸네일 이미지 중요, 중요내용 앞부분 배치
  1. 데이터를 기반으로 한 크리에이트브 전략 중요 전략 중요
  • 끊임없는 A/B 테스트
  • 최적 시간대 고려
  • 다이나믹 크리에이티브 ( 최상의 조합으로 게시가능)
  • 다이나믹 랭귀지

예시 : Seasonal Event 활용한 크리에이티브 전략

Pre-Holiday > During-Holiday > Post Holiday

  • Pre-Holiday

    1. 캠페인 목표 : 브랜드 인지도, 신규 고객 니즈 생성

    2. 크리에이티브 : 시선끄는,시각적 ,정보전달하는

    3. 추천하는 광고 포맷 : 영상, 캐러샐, 캔버스

  • During-Holiday

    1. 캠페인 목표 : 신규 고객 전환, 유효 고객 (최근 구매자, 웹사이트 방문자)

    2. 크리에이티브 : CTA 포함 직접적인 행동 유도, 상품 카탈로그

    3. 추천하는 광고 포맷 : 다이나믹 광고, 영상, 캔버스(카탈로그 포함)

  • Post-Holiday

    1. 캠페인 목표 : 비 전환 고객 유도, retention rate 증대

    2. 크리에이티브 : 브랜드 이미지 각인, 상품 최신 정보 제공, 고객 친밀도 형성

    3. 추천하는 광고 포맷 : 캐러샐, 다이나믹 광고, DR 이미지 광고

    $$CLV : (M-c)/1-r+i -AC$$

    • M : 고객 1인당 평균 매출 (1년단위)
    • c : 고객 1인당 평균 비용 ( 1년단위)
    • r : 고개 유지 비율 (retention rate)
    • i : 할인율
    • AC : 고객 획득 비용(Acquisition Cost)
    • 구매빈도 / 지출금액/ 지출예상금액/ 유지 잠재적 기간 고려
