예시 : Seasonal Event 활용한 크리에이티브 전략

Pre-Holiday > During-Holiday > Post Holiday

  • Pre-Holiday

    1. 캠페인 목표 : 브랜드 인지도, 신규 고객 니즈 생성

    2. 크리에이티브 : 시선끄는,시각적 ,정보전달하는

    3. 추천하는 광고 포맷 : 영상, 캐러샐, 캔버스

  • During-Holiday

    1. 캠페인 목표 : 신규 고객 전환, 유효 고객 (최근 구매자, 웹사이트 방문자)

    2. 크리에이티브 : CTA 포함 직접적인 행동 유도, 상품 카탈로그

    3. 추천하는 광고 포맷 : 다이나믹 광고, 영상, 캔버스(카탈로그 포함)

  • Post-Holiday

    1. 캠페인 목표 : 비 전환 고객 유도, retention rate 증대

    2. 크리에이티브 : 브랜드 이미지 각인, 상품 최신 정보 제공, 고객 친밀도 형성

    3. 추천하는 광고 포맷 : 캐러샐, 다이나믹 광고, DR 이미지 광고

    $$CLV : (M-c)/1-r+i -AC$$

    • M : 고객 1인당 평균 매출 (1년단위)
    • c : 고객 1인당 평균 비용 ( 1년단위)
    • r : 고개 유지 비율 (retention rate)
    • i : 할인율
    • AC : 고객 획득 비용(Acquisition Cost)
    • 구매빈도 / 지출금액/ 지출예상금액/ 유지 잠재적 기간 고려

  • 픽셀을 설치하면 뭐가 좋을까?

  1. 타겟팅 : 전환한 사람 토대로 타겟 생성하여 리마케팅

  2. 전환 추적 : 가입,장바구니, 구매 등 결과 측정

  3. 최적화 : 전환에 필요한 광고 최적화 진행(앱 다운로드를 가장 많이 할 것 같은 사람)

  • 픽셀의 기본 구조

  • 기본코드(타겟팅)

    단순 이벤트 추적이면 직접 넣을 수 있음

  • 이벤트코드(추적 및 최적화)

    페이지 depth가 복잡하고 다양한 이벤트 추적하고 싶다면 내용정리해 개발자에게 요청

  • 표준 이벤트 설치하기(웹 사이트 내 어떤 이벤트(전환)를 추적할 것인가?)

  • 픽셀 정리

    1. 웹사이트 원번 코드 사이에 넣기

    2. Facebook 기본 픽셀코드

    3. 표준 이벤트 코드

    4. 더 deep하게 보고 싶다면 ? 맞춤 전환 기반 이벤트 설정(계정당 100개)

    url 기반 + 이벤트 기반

    광고 관리자 > 이벤트 관리자 > 픽셀에서 확인 가능

  • 타겟은 무엇인가요?

페이스북 타겟 = 핵심타겟 + 맞춤 타겟 + 유사타겟

  • 핵심타겟

    페이스북 유저가 직접 입력한 정보와 페이스북에서 보이는 행동 기반 생성 타겟 마켓

    인구통계학적 특징(연령,언어,직장,업종 등) / 관심사(학력,비즈니스,테크놀로지,피트니스 등)

  • 맞춤타겟

    고객 개인정보 해치지 않는 범위에서 다양한 소서를 기반으로 특정한 고객군을 모아놓은 타겟

    ex. 페이지 왔었던 사람들, 구매하려다 만 사람들, 구매 경험있어서 이메일 가지고 있는 고객

    픽셀이 설치된 상태에서 사용 가능

    광고관리자 > 자산 > 타겟에서 생성가능

    맞춤 타겟 소스 1)고객리스트 2) 웹사이트트래픽 3)앱활동 4)참여

  • 유사타겟

    맞춤 타겟 특성 부석 기반 해당 국가 1-10% 범위로 확장한 타겟

    기존 고객 데이터 혹은 맞춤타겟을 활용하여 비슷한 새로운 타겟 찾을 수 있음

    무모하지 않게 신규 모수 확보할때 유리

    1. 소스 : 어떤 소스와 유사한 유저 ?

    2. 위치 : 어떤 국가 내 ?

    3. 타겟규모 : 생성할 유사 모수 크기 ? (무조건 1%가 좋은 것은 아님, 테스트 후 적용 필요)

      유사 %가 낮을수록 가장 유사한 특성 / %가 높을 수록 기준되는 맞춤 타겟

  • 콘텐츠 마케팅의 캠페인 기획 및 운영 프로세스

    1. 타겟 페르소나 정의 + 오디언서 맵핑(Targeting) : 타겟들의 행동 프로파일링 특징 정의

    4. 콘텐츠 생산 운영 : 스토리 보드와 가이드 먼저 컨펌 후 제작

    5. 콘텐츠 유통 및 발행 : 발행 일자도 미리 체크 (직장인 기준 2-3시에 sns 빈도 多)

    6. 분석 및 프로그램 최적화 : 다시 운영프로세스 반영하는 선순환

4. 퍼포먼스 마케팅

  • 마케팅 믹스 ( 노출 / 유입 / 전환)

  • 디지털 매체 믹스 : 여러 광고 매체를 섞어 예상 견적 파악, 광고 효과 예측할 수 있도록 도와주는 견적서 형태의 문서형태 + 적합 광고 상품과 그 특징을 파악하는 공부 필요

    1. 내가 속한 업종에서 많이 집행하고 있는 광고상품

    2. 나의 주요 타겟에게 광고가 노출될 수 있는 상품

    3. 내가 목표한 수치를 달성할 수 있는 광고상품

    4. 소재유형이나 노출영역, 광고속성과 같은 기준확인 : 머리 속 말고 명확한 가이드라인 필요

    ++ 특정 광고 중 목표 클릭 수 달성 되지 않는다면 광고 소재 변경 또는 타겟 범위 넓히거나 시너지 나는 상품을 추가하는 등의 대책 세움

    ++ 목표한 데이터 달성에 초점 맞춰 효율적 운영 목표

  • 광고형태 용어 체크리스트

    • 디스플레이광고(DA : Display Ads)

      웹사이트에 이미지 형태로 노출되는 광고 , 가장 흔하고 띠/박스 형태의 이미지로 노출

    • 검색광고(SA : Search Ads)

      특정 키워드 검색 시 상하단에 뜨는 광고

    • 네이티브 콘텐츠 광고(NA : Native Ads)

      배너광고 중 하나의 형태로 광고시청자의 광고 수용도가 높아 반응률 높은 편

  • 매체믹스 용어 체크리스트 (과금방식)

    • CPM(Cost Per Mile)

      광고 1000회 노출시키는데 따른 가격 책정 방식

    • CPC(Cost Per Click)

      클릭 당 과금하는 방식

    • CPV(Cost Per View)

      조회 수 당 과금하는 방식

    • CPI(Cost Per Install)

      앱 다운로드 당 과금하는 방식

  • 광고보고서 용어관련 페크리스트(광고셋팅 및 성과지표 관련)

    • 랜딩페이지

      콘텐츠 내에서 링크와 버튼 클릭 시 뜨는 페이지

    • 게재빈도수

      동일 광고시청자에게 몇 회의 광고가 노출되는 지를 측정한 값

    • 노출(Impression)

      광고가 노출되는 수 (중복O) : 몇회 노출

    • 도달(Reach)

      광고가 노출되는 사람 수(중복 X) : 몇명 노출

    • 클릭(Click)

      광고 클릭되는 수

    • 전환(Conversion)

      유입된 소비자가 특정 목표행동 취하는 것 (구매외 다양한 것도 될 수 있음)

1. 브랜드마케팅

  • 브랜딩 - 다름 - 등대 : 자신의 아이덴티티, 시장에 있어야 하는 이유 완성해가는 과정

  • 마케팅 - 나음 - 더 밝게 : 남과 달라지기 위해 시장에 대응하기 위한 의사결정에 초점

    • Plan : 과거의 마케팅 실적의 리뷰, 마케팅 전략 계획, 마케팅 전술 계획
    • Do : 캠페인 활동 전개
    • Evaluate : 평가 및 효율분석 (with 퍼포먼스 마케터 & 대행사)

  • 전략과 전술은 어떻게 다를까 ?

    • 전략적 계획 : 목표 설정 / 예산계획/ 컨셉&메세지
    • 전술적 계획 : 구체화된 전략을 어떻게 전개할 것인지 (브랜디드? 네이티브 ?)
    • 평가 및 반영 : 전체평가 실시 후 다음계획에 반영

  • 어느 부분이 좀 더 강점을 가지고 일 하는가 ?

    • 기획 전달 정보를 우선순위로 정리, 창의력 보다는 정리가 우선 > why 부터 생각

    • 마켓 리더와 유사점(Point of Parity) 내세워 속한 카테고리 알린 후,

      우리 브랜드 만의 차별점(Point of Difference) 인식

    • ex)

      비타500 - POP 피로해소 / POD 아이돌 모델, 노카페인

      이케아 POP 모던 가구 / POD DIY 형태, 탁이소&푸드코트

      러쉬 POP 자연주의 표방 목욕용품 / POD 강렬한 향, 컬러

++ POP와 POD를 찾는 것은 마케팅에서 중요한 부분이며, 혼재 되어있을 가능성이 높기 때문에 차별점에 대해 살펴보고 점검하는 것은 중요 ( 이전에는 차별점이 었으나 현재는 아닐 수도 있음)
